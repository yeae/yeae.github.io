RustDesk 自建服务器完整指南

什么是 RustDesk？

RustDesk 是一款开源的远程桌面软件，使用 Rust 语言编写，可以作为 TeamViewer 和 AnyDesk 的开源替代品。它支持 Windows、macOS、Linux、Android 和 iOS 等多个平台。

为什么选择自建服务器？

  • 隐私安全：数据完全掌控在自己手中
  • 无使用限制：不受官方服务器限制
  • 速度更快：可部署在离用户更近的服务器
  • 完全免费：无需付费订阅

服务器要求

最低配置

  • CPU: 1核
  • 内存: 512MB
  • 硬盘: 10GB
  • 带宽: 1Mbps（建议 5Mbps 以上）
  • 操作系统: Linux (Ubuntu 20.04/22.04 推荐)

端口要求

需要开放以下端口：

  • 21115 TCP (hbbs - RustDesk ID注册服务器)
  • 21116 TCP/UDP (hbbs - NAT类型测试)
  • 21117 TCP (hbbr - RustDesk 中继服务器)
  • 21118 TCP (hbbs - Web客户端)
  • 21119 TCP (hbbs - Web客户端)

常见问题

1. 无法连接到服务器

检查项：

  • 服务器防火墙端口是否开放
  • 云服务商安全组规则是否配置
  • 服务是否正常运行：systemctl status rustdesk-hbbs rustdesk-hbbr
  • 客户端配置是否正确

2. Key 验证失败

  • 确保复制完整的公钥内容
  • 去除所有空格和换行符
  • 确认服务端和客户端的密钥匹配

3. 连接速度慢

  • 检查服务器带宽
  • 尝试使用 UDP 中继
  • 考虑升级服务器配置

4. 查看日志
