Author: yeae Date: May 30, 2023  19:47:22

windows的命令行（cmd），输入conda env list 查看已安装环境

如何修改虚拟环境安装位置
查看一下conda信息，命令行输入

conda info

修改C:\Users\wader下 .condarc（在最下面）

show_channel_urls: true

envs_dirs:

  • D:\Instance\deep_learning\Virtual_Environment\envs
    pkgs_dirs:
  • D:\Instance\deep_learning\Virtual_Environment\pkgs
    channels:
  • https://mirrors.bfsu.edu.cn/anaconda/cloud/pytorch/
  • https://mirrors.ustc.edu.cn/anaconda/pkgs/free/
  • https://mirrors.ustc.edu.cn/anaconda/pkgs/main/
  • defaults

创建虚拟环境

1 在命令行中输入下列指令创建虚拟环境

conda create -n opencv python==3.8.5

2 输入下列指令激活虚拟环境，出现下图所示的小括号表示环境激活成功

conda activate opencv

3 安装程序所需要的包

pip3 install opencv-python
pip3 install torch torchvision torchaudio https://pytorch.org/get-started/locally/
pip install tensorflow==2.3.0 https://www.tensorflow.org/install?hl=zh-cn
